#### 1543_문서 검색

>  문제 분석 
>
> - 특정 단어가 총 몇번 등장하는지 파악 
> - 단 중복 세기는 금지 

> 코드 설계 
>
> - 방법1: 0부터 len(word-1)까지 다 돌면서 해당 단어가 시작하는 idx 저장 
> - 방법2: 해당 단어가 있는 위치를 다 저장한 후, start는 늦게, end는 빠르게 되는 지점들 갯수를 센다 
> - 방법 2 정답!!!! start는 늦게 end는 이르게가 답의 핵심!!

훨씬 쉬운 남의 코드 ...

```python
a = input()
b = input()
cnt = 0
n = 0
while n <= len(a) - len(b):	# index error 해결 
    if a[n:n + len(b)] == b:	# 
        cnt += 1
        n += len(b)
    else:
        n += 1
print(cnt)
```
